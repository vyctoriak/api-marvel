## Getting Started

### Instalando dependências

Para instalar os pacotes e rodar na sua máquina, siga as intruções abaixo:

```
npm install
```

ou 

```
npm i
```

E em seguida:

```
npm start
```

Você também pode ver o projeto rodando na URL: https://vigorous-mccarthy-233f0b.netlify.app/
